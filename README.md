# TPS-Online
* This Python package, named TPS-Online, corresponds to the code of “ Online PU Learning with Truncated Propensity Scores”
* 
* Positive-unlabeled (PU) learning often suffers from extreme propensity scores under strong labeling bias, leading to unstable model training. To address this issue, we propose a truncation framework that bounds the propensity scores within a reasonable range, and design a robust online learning algorithm (TPS-Online) with a Sigmoid-weighted loss. Theoretically, TPS-Online achieves sublinear regret bounds under mild assumptions. Empirically, we validate its effectiveness on both simulated and real-world datasets.
* The repository contains two subfolders, corresponding to experiments under simulation dataset and real_world datasets.
